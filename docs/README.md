# GreatResetFantasy iOS App

## Steps to run the app

1. **Configure Supabase credentials** (required before first build):
   ```bash
   cp docs/Supabase_credentials_setup.xcconfig Supabase/Secrets.xcconfig
   ```
   Edit `Supabase/Secrets.xcconfig` and add your Supabase URL and anon key from **Supabase Dashboard → Settings → API**.  
   `Supabase/Secrets.xcconfig` is gitignored and should never be committed.

2. Build & run the app.
